# insurance-loss-simulation
Monte Carlo simulation of insurance claims and loss thresholds

## Problem
Simulate insurance claims and estimate the probability that total losses exceed a specified threshold.

## Approach
Modeled claim frequency using Bernoulli trials and claim severity using a uniform distribution. Ran multiple simulations of a 1,000-policy portfolio and tracked total losses per simulation.

## Tools Used
- Python
- random module
- loops and conditional logic

## Key Result
Estimated probability that total losses exceed $125,000 is approximately X% (based on simulation).

## How to Run
python simulation.py
